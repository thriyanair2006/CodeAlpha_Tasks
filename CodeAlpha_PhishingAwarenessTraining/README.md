# Phishing Awareness Training

## Project Overview

This project was created as part of the **CodeAlpha Cyber Security Internship** to spread awareness about phishing attacks and online security.

## Topics Covered

* What is Phishing?
* Types of Phishing Attacks
* Recognizing Phishing Emails
* Identifying Fake Websites
* Social Engineering Techniques
* Real-World Phishing Examples
* Cybersecurity Best Practices
* Interactive Quiz

## Tools Used

* Microsoft PowerPoint
* Visual Studio Code

## Key Learning Outcomes

* Identify phishing attempts
* Recognize suspicious emails and websites
* Understand social engineering tactics
* Apply safe online practices

## Conclusion

Phishing attacks continue to be one of the most common cyber threats. By staying alert, verifying information, and following cybersecurity best practices, users can protect themselves from becoming victims.
